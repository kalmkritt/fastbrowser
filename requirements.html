system requirements

operating system
Windows 10 or later (64-bit).
memory
minimum 2GB RAM. 4GB+ recommended for smooth performance with multiple tabs.
storage
approximately 200MB disk space for installation.
processor
Intel or AMD 64-bit processor with SSE2 support.
graphics
dedicated or integrated GPU recommended for hardware acceleration. integrated graphics work fine.
installation
no admin rights required to run fastbrowser. portable installation means no registry modifications.
tested on
developed and tested on Lenovo IdeaPad 5 2-in-1 14AHP9 (Gen 9). achieves the stats on the home page on mid-range hardware.
